# documentacion

## resumen

este proyecto publica una app web con:

- frontend en react y vite
- backend en node.js, express y mongodb
- autenticacion con jwt
- contrasenas protegidas con `crypto.scrypt`

ligas activas:

- frontend: `https://web-practicas-dev.vercel.app`
- api: `https://web-practicas-dev-api.onrender.com`

## indice

- `local.md`: ejecucion local y validacion
- `api.md`: autenticacion, roles y endpoints
- `deploy.md`: despliegue real en render, vercel y atlas
- `entrega.txt`: evidencia corta para entregar
